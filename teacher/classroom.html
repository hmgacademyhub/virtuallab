📚 Classroom Management

Create and manage classes, import students, and organize your virtual laboratory.

📖 About Classroom Management

The Teacher Dashboard (accessible from the main dashboard page) provides full classroom management features:

  • Create Classes: Name your class, assign a teacher, and generate a unique access code.
  • Add Students: Add students individually or import in bulk via CSV.
  • Class Codes: Share the unique class code with students for self-registration.
  • Assignments: Assign specific simulations to classes with due dates.
  • Analytics: View class-wide progress, completion rates, and performance trends.
