Interactive Periodic Table

60+ Curriculum-Aligned Sims

10 per subject across Physics, Chemistry, Biology, Mathematics, General Science. WAEC/NECO/JAMB/IGCSE mapped.

Gamification

10 badges, leaderboard, points/levels. Complete sims (+10pts), score 100% on quizzes (+25pts), earn badges (+50pts).

Teacher Dashboard

Create classes, manage students, track progress. All with LocalStorage — no server needed, complete privacy.

Student Portal

Track progress, earn badges, climb the leaderboard. Complete simulations to earn points and level up.

Parent Portal

Monitor learning progress — completed simulations, quiz scores, badges earned, time spent learning.

Offline-First

Service Worker caching. ~300KB total. Works on 2G and low-end devices. Built for Nigerian realities.

Enterprise Features (Free)

School-wide deployment, admin dashboard, bulk CSV import, custom branding — all zero cost.
